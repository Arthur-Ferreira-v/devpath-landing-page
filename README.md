# DevPath — Landing Page

A modern and responsive landing page for **DevPath**, a fictional platform that helps beginner developers build their portfolio step by step.

## 🚀 Features

- Responsive Navbar with smooth hover effects
- Hero Section with gradient text animation
- Features Section with interactive cards
- Built with pure HTML, CSS and JavaScript

## 🛠️ Technologies

- HTML5
- CSS3 (Flexbox, Grid, Animations)
- JavaScript (coming soon)

## 📚 Concepts Practiced

- CSS Flexbox and Grid
- Responsive Design
- CSS Animations and Transitions
- CSS Custom Properties
- BEM-like class naming
