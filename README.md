# DevPath — Landing Page

A modern and responsive landing page for **DevPath**, a fictional platform 
that helps beginner developers build their portfolio step by step.

## 🌐 Live Demo

👉 [https://arthur-ferreira-v.github.io/devpath-landing-page/](https://arthur-ferreira-v.github.io/devpath-landing-page/)

## 🚀 Features

- Responsive Navbar with smooth hover effects
- Hero Section with gradient text animation
- Features Section with interactive cards
- Testimonials Section
- CTA Section
- Footer with links
- Hamburger Menu for mobile
- Fully responsive — mobile, tablet and desktop

## 🛠️ Technologies

- HTML5
- CSS3 (Flexbox, Grid, Animations, Media Queries)
- JavaScript (DOM Manipulation)

## 📚 Concepts Practiced

- CSS Flexbox and Grid
- Responsive Design — Desktop First approach
- CSS Animations and Transitions
- Media Queries and Breakpoints
- CSS Pseudo-selectors (nth-child, hover)
- JavaScript classList.toggle()
- Git and GitHub workflow
- GitHub Pages deployment
